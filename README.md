# Todo Application (CLI-Based) with Automated Testing

A Python-based command-line Todo application designed to demonstrate
**core logic validation, test automation, and QA best practices**.

---

## Overview

This project implements a simple Todo management system using Python,
focusing on:

- Clean business logic design
- Unit testing with pytest
- Test isolation and reliability
- Edge-case validation

---

## Features

- Add tasks
- View tasks
- Mark tasks as complete
- Delete tasks
- Persistent task handling (if applicable)

---

## Architecture

The application is structured into logical components:

- **Core Logic Layer**: Handles task behavior and state
- **CLI Interface**: Accepts user input and executes commands
- **Storage Layer**: Manages saving/loading tasks (if implemented)

---

## Running the Application

bash
python cli.py
